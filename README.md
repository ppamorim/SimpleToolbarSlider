# SimpleToolbarSlider

[![Build Status](https://api.travis-ci.org/ppamorim/SimpleToolbarSlider.svg?branch=master)](https://travis-ci.org/ppamorim/SimpleToolbarSlider)

However, I use Fresco to load the images from the web.
